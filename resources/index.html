/api/travelTimeTableData?{devices}&{date}

Parameters

{devices} Array of IDs; required

id: string, minimum two device IDs; required

Example: ['a83', 'hj72', 'w21']

{date} Object of start and end date; required

from: number - start date in milliseconds; required

to: number - end date in milliseconds; required

Example: {from: 1539129600000, to: 1539180000000}

/api/travelTimeTableData?devices=a83%2Chj72%2Cw21&date=%7B%22from%22%3A1539129600000%2C%22to%22%3A1539180000000%7D
